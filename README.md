What is it ?
=================

MaxRects算法是一个二维图像排列算法.MaxRects算法的目的是找到一个最佳的方式，到将许多矩形最优化放在到另一个较大的矩形里。


About the Demo
=================

仿照《找你妹》游戏制作的，在UIScrollView中加入各种图形。应用了MaxRects算法。
